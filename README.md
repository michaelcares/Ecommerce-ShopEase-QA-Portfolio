# Ecommerce-ShopEase-ManualQA-Portfolio

Manual QA testing project simulating a full test cycle
on a mobile e-commerce shopping application.

# What's inside
- Test Plan (scope, tools, entry/exit criteria)
- 30 test cases across 7 modules
- 8 detailed JIRA bug reports
- Test Summary Report with pass/fail metrics

# Tools used
JIRA · Google Sheets · Android Emulator

# Modules tested
Registration · Login · Product Search · Cart ·
Checkout · Order Tracking · Returns & Refunds
